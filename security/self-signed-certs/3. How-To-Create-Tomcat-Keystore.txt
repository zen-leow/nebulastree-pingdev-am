# first use OpenSSL to create a P12 bundle

openssl pkcs12 -export -in my-SSL.cer -inkey my-private.key -certfile Combined_CertChain.cer -out myP12.p12 -name tomcat

# then use keytool to convert it into a java keystore

keytool -importkeystore -srckeystore myP12.p12 -srcstoretype PKCS12 -destkeystore tomcat.keystore


Make sure all the passwords entered are the same for both commands.
